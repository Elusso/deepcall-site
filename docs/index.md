<h2>🧠 Наша экспертиза (на чём мы растем)</h2>

<p>Мы специализируемся на:</p>

<ul>
  <li><strong>Predictive Call Tracking</strong> — прогнозируем целевые отклики по длительности, частоте и таймингу звонков.</li>
  <li><strong>Reverse Telemetry De-anonymization</strong> — обратная деанонимизация: по номеру кандидата восстанавливаем список рекрутеров, которые ему звонили.</li>
  <li><strong>Voice AI Orchestration</strong> — оркестровка ИИ-агентов (clawd-ботов) с локальным LLM-стеком (Ollama + Whisper + XTTS).</li>
  <li><strong>SIP-интеграция любой сложности</strong> — поднимаем виртуальные АТС с динамической подстановкой масок номеров.</li>
  <li><strong>Аудит конкурентного эфира</strong> — пассивный сбор мета-данных о звонках других агентств без доступа к их CRM.</li>
  <li><strong>Мониторинг поведенческих маркеров кандидатов</strong> — анализируем частоту обновления резюме, геолокацию, время откликов.</li>
</ul>
